# university-test
Тестовое задание для курса "Хранилища данных. Введение в большие данные"
## Установка
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
## Запуск
### Сколько книг у каждого пользователя
`python cli.py library-users`
### Любимый жанр пользователя
`python cli.py users-best-genre --users user-1 --users user-2`
